# SaaSFlow Product Documentation

## 1. Core Concepts

### 1.1 Workspaces
The top-level container for all projects. Members are invited to workspaces.

### 1.2 Kanban Boards
Visual task management using columns (To Do, In Progress, Review, Done). Tasks move between columns via drag-and-drop.

### 1.3 Automations ("Flows")
"When X happens, do Y" logic. 
- *Free Plan*: 100 runs/month.
- *Pro Plan*: 5,000 runs/month.

## 2. Billing & Plans

### 2.1 Subscription Tiers
- **Free**: Up to 5 members, 1 project.
- **Pro**: $12/user/month. Unlimited projects, full automation suite.
- **Enterprise**: Custom pricing, SSO/SAML, dedicated manager.

## 3. Integrations
- **Slack**: Get notifications and create tasks with `/saasflow`.
- **GitHub**: Auto-link PRs to Kanban cards.

## 4. Troubleshooting
- **Sync Issues**: Hard refresh (Ctrl+F5) or check status page.
- **Access**: Reset password via the login screen link.
